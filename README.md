# Plexus Dental — Landing Page

A simple static landing page for a dentist website, built with **React** and **JavaScript** (no TypeScript).

## Run locally

```bash
npm install
npm run dev
```

Then open [http://localhost:5173](http://localhost:5173).

## Build for production

```bash
npm run build
```

Output is in the `dist/` folder. Serve it with any static host.

## Tech stack

- React 18
- Vite 5
- Plain JavaScript (JSX)
